# P&R Gestión

P&R Gestión is a web application developed as an academic project for managing work orders in a technical services company. The system allows registering clients and technicians, creating and tracking work orders, adding comments, searching orders, and generating activity reports.

## Main Features
- Technician login
- Client management
- Technician management
- Work order creation and tracking
- Status updates for orders
- Comments on work orders
- Search orders by number
- Activity reports by date range
- Server-side validation

## Technologies Used
- ASP.NET Web Forms
- C#
- HTML
- CSS
- Master Page structure

## Notes
This version uses in-memory lists instead of a database, following the project requirements.

## Project Purpose
This project was created as part of a Programming course at CEI - CLAEH to apply web development concepts using ASP.NET and C#.
